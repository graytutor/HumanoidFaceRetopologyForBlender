INSTALLATION
============

Install release zip
-------------------

1. Download the release zip, for example:

   HumanoidFaceRetopology_v1_0_0_release.zip

2. Open Blender.
3. Go to Edit > Preferences > Add-ons.
4. Click Install....
5. Select the release zip.
6. Enable Humanoid Face Retopology(HFR).
7. Open View3D > Sidebar > HFR.

Development install
-------------------

For development, use the source package under:

addon/HumanoidFaceRetopology

Copy or symlink that folder into Blender's add-ons directory, then enable the add-on.

In a development install, HFR_SHOW_DEV_OPTIONS may be set to True so the DevOption UI is available.

Release install behavior
------------------------

Release zips should be built with:

HFR_SHOW_DEV_OPTIONS = False

This hides the DevOption toggle and developer-only panels from normal users.
