CHANGELOG
=========

v1.0.0 - First public release
-----------------------------

- Initial public release of Humanoid Face Retopology(HFR).
- Includes the bundled default humanoid face retopology template.
- Includes the bundled default template binding data.
- Provides the main workflow: select Target Mesh, create and adjust landmarks, then generate retopology.
- Includes user-facing Advanced options for more detailed control when needed.
