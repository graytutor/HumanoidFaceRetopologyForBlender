TROUBLESHOOTING
===============

Template Mesh is missing
------------------------

Click Reload Default Template in 1. Setup.

If it still fails, check that the add-on package contains:

templates/HFRTemplate.blend
templates/HFRTemplateBinding.json

Template Binding is not OK
--------------------------

For normal users, reinstall the release zip or reload the default template.

For developers, enable DevOption and inspect DevOption: Template Binding.

Landmarks are hard to see
-------------------------

Enable Landmark to Front.

Some scalp/back landmarks intentionally do not always draw in front, because showing back-side landmarks through the head can make placement confusing.

Mirror does not behave as expected
----------------------------------

Check:

- The target is centered around world X = 0.
- The face is oriented consistently.
- Landmark Mirror X is enabled.
- Mirror direction is correct: L -> R or R -> L.

Generated result is distorted
-----------------------------

First adjust landmarks and regenerate.

Common causes:

- Center line landmarks are off-axis.
- Eye or mouth landmarks do not follow the actual loop boundary.
- Ear landmarks are pulled too far from the intended ear structure.
- Target mesh has unapplied rotation/scale.
- Target mesh orientation does not match the expected axes.

Snap result looks too aggressive
--------------------------------

Enable Advanced and inspect:

- Snap Strength
- Max Dist
- Protect Anchors
- Ear Guard
- Eye Guard

Change one setting at a time and regenerate.
